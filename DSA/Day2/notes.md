# Day 2 — Operators in Java

## 📚 Topics Covered
- Types of Operators in Java
- Arithmetic Operators
- Unary Operators
- Relational Operators
- Logical Operators
- Assignment Operators

## 🧠 Key Learnings

**Arithmetic Operators** — `+`, `-`, `*`, `/`, `%`
- Used for basic mathematical operations
- `%` gives the remainder of division

**Unary Operators** — `+`, `-`, `++`, `--`, `!`
- Operate on a single operand
- `++a` (pre-increment) vs `a++` (post-increment) — order matters!

**Relational Operators** — `==`, `!=`, `>`, `<`, `>=`, `<=`
- Always return a boolean (`true` or `false`)
- Used for comparisons

**Logical Operators** — `&&`, `||`, `!`
- `&&` — true only if both conditions are true
- `||` — true if at least one condition is true
- `!` — reverses the boolean value

**Assignment Operators** — `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- Shorthand for arithmetic + assignment together
- e.g. `a += 5` is same as `a = a + 5`
