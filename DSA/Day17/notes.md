## 💻 Project — EcoStride 🌿

A **Real-Time Environmental Health & Safe Routing Platform**

* Built on a problem statement from **SIH 2025 (Govt of Delhi)** — Air Quality Monitoring & Eco-Friendly Routing.
* Developed in a few weeks by **Team-X (5 members)**.
* My Role:

  * UI/UX Developer
  * Live Email Integration Lead

### ⚙️ Tech Stack

* Frontend → React (Mobile + Web)
* Backend → Python + Flask
* Maps & Routing → Leaflet + OSRM
* Data → OpenWeatherMap, AQICN, Govt APIs

### 🔑 Key Features

* Live Safe Routing → Routes ranked from cleanest to most polluted
* Colony-wise AQI → Localized air quality data
* AI Chatbot → Health & route suggestions
* Email Alerts → AQI threshold notifications
* 72-Hour Forecast → AQI prediction
* Policy Simulation → Govt intervention impact analysis
