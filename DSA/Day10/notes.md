## 🧠 Key Learnings

### Introduction to Sorting

Sorting is the process of **arranging elements of a list or array in a specific order**, usually in **ascending or descending order**.

Sorting makes it easier to:

* Search elements efficiently
* Organize data
* Apply many algorithms like **Binary Search**

Example:

Unsorted array

```
5 2 9 1 3
```

Sorted array (Ascending)

```
1 2 3 5 9
```

---

### Overview of Different Sorting Techniques

There are many algorithms used to sort data, each with different efficiency and use cases.

Some common sorting algorithms include:

* **Bubble Sort**
  Repeatedly compares adjacent elements and swaps them if they are in the wrong order.

* **Selection Sort**
  Finds the minimum element from the unsorted part and places it at the beginning.

* **Insertion Sort**
  Builds the sorted array one element at a time by inserting elements in their correct position.

* **Merge Sort**
  Uses the **divide and conquer** approach by splitting the array and merging sorted parts.

* **Quick Sort**
  Also uses divide and conquer but selects a **pivot element** and partitions the array around it.

Each sorting algorithm differs in:

* **Time Complexity**
* **Space Usage**
* **Stability**
* **Best use cases**
