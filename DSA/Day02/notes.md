# Day 2 — Operators, Conditionals & Loops in Java

## 📚 Topics Covered
- Types of Operators (Arithmetic, Unary, Relational, Logical, Assignment)
- Conditional Statements (if-else, Ternary, Switch)
- Loops (While, For, Do-While)

## 🧠 Key Learnings

### ⚙️ Operators
**Arithmetic** — `+`, `-`, `*`, `/`, `%`
- `%` gives remainder of division

**Unary** — `++`, `--`, `!`
- `++a` (pre-increment) vs `a++` (post-increment) — order matters!

**Relational** — `==`, `!=`, `>`, `<`, `>=`, `<=`
- Always return boolean (`true` or `false`)

**Logical** — `&&`, `||`, `!`
- `&&` — true only if both conditions are true
- `||` — true if at least one condition is true

**Assignment** — `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- e.g. `a += 5` is same as `a = a + 5`

---

### 🔀 Conditional Statements
**if-else** — executes block based on condition

**Ternary Operator** — shorthand for if-else → `condition ? true : false`

**Switch** — cleaner alternative to multiple if-else chains

---

### 🔁 Loops
**While loop** — runs while condition is true (condition checked first)

**For loop** — best when number of iterations is known

**Do-While loop** — runs at least once (condition checked after)
