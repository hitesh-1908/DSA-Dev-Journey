## 🧠 Key Learnings

* Continued learning **OOPS concepts** in Java.
* Learned about **destructors** (though Java handles memory using garbage collection, so destructors are not explicitly used like in C++).
* Understood **inheritance**, where one class acquires properties and methods of another class.
* Studied different **types of inheritance**:

  * Single Inheritance
  * Multilevel Inheritance
  * Hierarchical Inheritance
* Learned about **polymorphism**, where the same function behaves differently in different situations.
* Understood **method overloading** (same method name, different parameters — compile-time polymorphism).
* Learned **method overriding** (child class provides its own implementation of a parent class method — runtime polymorphism).
* Studied **packages in Java**, which help organize classes and avoid naming conflicts.
