# Day 15 — Pattern Practice

## 📚 Topics Covered

* Normal Patterns
* Advanced Patterns
* Pattern logic using nested loops
* Space and character management in patterns

## 🧠 Key Learnings

* Today I spent time **practicing both normal and advanced pattern problems** to strengthen my understanding of loops and pattern logic.
* Pattern problems mainly rely on **nested loops**, where the outer loop controls the number of rows and the inner loops control spaces, stars, or numbers printed in each row.
* I revised how different patterns are formed by carefully managing **spaces, characters, and row/column relationships**.
* While practicing, I focused on understanding the **logic behind the patterns instead of memorizing the code**.
* I noticed that most pattern questions follow similar logic such as:

  * Managing **spaces before characters**
  * Printing **increasing or decreasing sequences**
  * Using **conditions to create hollow patterns**
  * Dividing complex patterns into **upper and lower halves**.
* Practicing these patterns helped reinforce my understanding of **loop flow, index control, and pattern symmetry**.
* I also revised patterns I had previously solved such as **pyramids, inverted pyramids, hollow rectangles, rhombus patterns, butterfly patterns, and diamond patterns**.
* This practice helped improve my ability to **visualize patterns and translate them into code using loops**.
