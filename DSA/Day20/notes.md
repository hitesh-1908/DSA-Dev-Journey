# Day 20 — OOPS

## 📚 Topics Covered

* Classes & Objects
* Access Modifiers
* Getters and Setter Methods
* Encapsulation
* Constructors
* Types of Constructors
* Copy Constructor

## 🧠 Key Learnings

* Today I started **Object-Oriented Programming (OOPS)** in Java.
* Learned how **classes act as blueprints** and objects are instances of those classes.
* Understood different **access modifiers** like:

  * `public`
  * `private`
  * `protected`
  * `default`
* Learned about **getters and setters**, which are used to access and update private variables safely.
* Understood **encapsulation**, where data and methods are wrapped inside a single unit and direct access is restricted.
* Studied **constructors**, which are special methods used to initialize objects.
* Learned different **types of constructors**:

  * Default Constructor
  * Parameterized Constructor
* Also learned about **copy constructor**, which is used to create a new object by copying values from another object.
